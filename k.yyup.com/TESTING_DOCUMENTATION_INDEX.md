## 🎯 **测试执行指南**

### 🚀 **快速开始**

#### **前端测试**
```bash
# 进入前端目录
cd client

# 运行所有测试
npm test

# 运行特定类型测试
npm run test:unit           # 单元测试
npm run test:integration    # 集成测试
npm run test:e2e           # E2E测试
npm run test:coverage      # 覆盖率测试
```

#### **后端测试**
```bash
# 进入后端目录
cd server

# 运行所有测试
npm test

# 运行特定类型测试
npm run test:unit           # 单元测试
npm run test:integration    # 集成测试
npm run test:models        # 模型测试
npm run test:controllers   # 控制器测试
```

#### **全项目测试**
```bash
# 在项目根目录
npm run test:all           # 运行所有测试
npm run validate          # 代码验证 (TS+ESLint+测试)
```
