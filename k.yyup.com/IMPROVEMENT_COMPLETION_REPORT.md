## 💻 技术实现

### 新增方法
```typescript
// 填充示例
function fillExample(example: string) {
  prompt.value = example;
}

// 清空表单
function clearForm() {
  prompt.value = '';
  selectedDomain.value = 'science';
  ageGroup.value = '4-5岁';
}
```
