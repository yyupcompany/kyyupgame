## 🎨 色彩方案

### 主色系（幼儿园友好）

```
主色：#FF6B6B (活力红)
辅色：#4ECDC4 (清爽青)
强调色：#FFE66D (温暖黄)
背景色：#F7F9FC (柔和白)
文字色：#2C3E50 (深灰)
```

### 色彩应用

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主按钮 | #FF6B6B | 生成、保存等主要操作 |
| 次按钮 | #4ECDC4 | 编辑、预览等次要操作 |
| 进度条 | #FFE66D → #FF6B6B | 渐变进度显示 |
| 背景 | #F7F9FC | 整体背景 |
| 卡片 | #FFFFFF | 内容卡片 |
| 文字 | #2C3E50 | 主文字 |
| 辅文字 | #7F8C8D | 说明文字 |

## 🎯 交互流程设计

### 流程1：生成课程

```
1. 教师输入
   ↓
2. 点击"生成课程"按钮
   ↓
3. 显示进度面板
   ├─ 代码生成进度
   ├─ 图片生成进度
   └─ 视频生成进度
   ↓
4. 资源加载完成
   ↓
5. 显示预览界面
   ├─ 左侧图片轮播
   └─ 右侧视频播放
```

### 流程2：图片轮播

```
用户点击 ◀/▶
   ↓
切换图片
   ↓
更新图片描述
   ↓
更新导航指示器
```

### 流程3：视频播放

```
用户点击播放
   ↓
视频开始播放
   ↓
显示进度条
   ↓
用户可以暂停、快进、调整音量
```

## 🎨 动画效果

### 1. 进度条动画

```css
/* 渐变色进度条 */
background: linear-gradient(90deg, #FFE66D, #FF6B6B);
animation: progress-flow 2s infinite;
```

### 2. 图片切换动画

```css
/* 淡入淡出效果 */
transition: opacity 0.3s ease-in-out;
```

### 3. 按钮悬停效果

```css
/* 缩放 + 阴影 */
transform: scale(1.05);
box-shadow: 0 4px 12px rgba(255, 107, 107, 0.3);
```

### 4. 加载动画

```css
/* 旋转加载 */
animation: spin 1s linear infinite;
```

这个UI设计方案的特点：

✅ **响应式**：支持桌面、平板、手机
