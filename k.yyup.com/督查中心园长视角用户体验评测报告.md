### ⚠️ 可优化点

**园长痛点**: "每次检查的项目其实是固定的（环境、消防、卫生...），现在每次都要手动添加，很麻烦"

**建议**：
```vue
<!-- 新增：从检查类型加载模板 -->
<el-button @click="loadTemplateItems">
  从检查类型加载标准检查项
</el-button>

// 自动加载检查类型中定义的标准检查项
const loadTemplateItems = () => {
  const standards = planData.inspectionType?.standards || [];
  formData.items = standards.map(std => ({
    itemName: std.item,
    itemCategory: '标准检查项',
    status: 'pass',
    score: 0,
    maxScore: std.score,
    problemDescription: '',
    photos: [],
    notes: ''
  }));
};
```

**预期效果**: 点一下按钮，10个检查项自动加载完成，只需要填结果

#### 2. 快捷评分功能缺失

**园长场景**: "大部分检查项都是满分通过，现在要一个个输入分数"

**建议**：
```vue
<!-- 快捷操作 -->
<el-button @click="markAllPass" size="small">
  全部通过（满分）
</el-button>

const markAllPass = () => {
  formData.items.forEach(item => {
    item.status = 'pass';
    item.score = item.maxScore;
  });
};
```

#### 3. 自动计算总分缺失

**园长痛点**: "填完所有检查项，还要自己算总分，容易算错"

**建议**：
```vue
<!-- 自动计算总分 -->
<el-form-item label="总分">
  <el-input-number v-model="autoCalculatedScore" disabled />
  <el-button @click="useCalculatedScore" type="text">
    使用计算值
  </el-button>
</el-form-item>

// 自动计算
const autoCalculatedScore = computed(() => {
  return formData.items.reduce((sum, item) => sum + (item.score || 0), 0);
});
```
