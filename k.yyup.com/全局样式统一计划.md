## 🎯 核心设计令牌

### 颜色系统
```scss
// 主色调
--primary-color: #6366f1;        // 紫蓝色
--primary-hover: #4f46e5;        // 深紫蓝
--primary-light: #818cf8;        // 浅紫蓝

// 功能色
--success-color: #10b981;        // 绿色
--warning-color: #f97316;        // 橙色
--danger-color: #ef4444;         // 红色
--info-color: #06b6d4;           // 青色

// 背景色
--bg-primary: #0f172a;           // 主背景
--bg-secondary: #1e293b;         // 次背景
--bg-tertiary: #334155;          // 三级背景
--bg-card: #1e293b;              // 卡片背景

// 文字色
--text-primary: #f8fafc;         // 主文字
--text-secondary: #cbd5e1;       // 次文字
--text-muted: #94a3b8;           // 弱文字
```

### 间距系统
```scss
--spacing-xs: 0.25rem;   // 4px
--spacing-sm: 0.5rem;    // 8px
--spacing-md: 0.75rem;   // 12px
--spacing-lg: 1rem;      // 16px
--spacing-xl: 1.5rem;    // 24px
--spacing-2xl: 2rem;     // 32px
```

### 字体系统
```scss
--text-xs: 0.75rem;      // 12px
--text-sm: 0.875rem;     // 14px
--text-base: 1rem;       // 16px
--text-lg: 1.125rem;     // 18px
--text-xl: 1.25rem;      // 20px
--text-2xl: 1.5rem;      // 24px
--text-3xl: 1.875rem;    // 30px
```

### 阴影系统
```scss
--shadow-sm: 0 1px 2px 0 rgba(0, 0, 0, 0.3);

```

## 🔧 技术规范

### 容器类使用规范
```vue
<template>
  <div class="center-container">
    <!-- 页面内容 -->
  </div>
</template>

<style scoped lang="scss">
.center-container {
  // 使用 CSS 变量，不硬编码
  background: var(--bg-secondary);
  padding: var(--spacing-lg);
  // ...
}
</style>
```

### 颜色使用规范
```scss
// ❌ 禁止
color: #f8fafc;
background: #1e293b;

// ✅ 推荐
color: var(--text-primary);
background: var(--bg-secondary);
```

### 间距使用规范
```scss
// ❌ 禁止
padding: 24px;
margin: 16px;

// ✅ 推荐
padding: var(--spacing-xl);
margin: var(--spacing-lg);
```
