# 课程生成器改进 - 测试指南

## 🚀 快速开始

### 1. 启动应用

```bash
# 在项目根目录
npm run start:all

# 或分别启动
npm run start:frontend  # 前端：http://localhost:5173
npm run start:backend   # 后端：http://localhost:3000
```

### 2. 访问页面

打开浏览器访问：
```
http://localhost:5173/teacher-center/creative-curriculum/interactive
```
