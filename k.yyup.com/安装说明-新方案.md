# WSL Ubuntu安装新方案

看来直接使用桌面版ISO文件不能正确导入WSL。我建议使用以下方案：

## 方案1：使用Microsoft Store（推荐）

1. 打开Microsoft Store
2. 搜索"Ubuntu"
3. 安装"Ubuntu 22.04.2 LTS"
4. 等待下载和安装完成

## 方案2：使用命令行安装

```powershell
# 安装Ubuntu 22.04 LTS
wsl --install -d Ubuntu-22.04
```

## 安装完成后：

1. 安装桌面环境：
```bash
# 进入WSL
wsl

# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装XFCE桌面
sudo apt install -y xfce4 xfce4-goodies

# 安装远程桌面服务
sudo apt install -y xrdp
sudo systemctl enable xrdp
sudo systemctl start xrdp
```

2. 连接到桌面：
- 按 Win + R
- 输入：mstsc
- 连接：localhost:3389

您想使用哪种方案？我建议使用方案1（Microsoft Store），这是最稳定可靠的方式。
