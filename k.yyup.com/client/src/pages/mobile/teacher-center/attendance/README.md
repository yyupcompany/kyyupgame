# 移动端教师考勤管理页面

## 页面概览

移动端教师考勤管理页面是PC端教师考勤功能的完整移植，专为移动设备优化设计，提供完整的考勤管理功能。

## 文件结构

```
attendance/
├── index.vue                       # 主页面，包含Tab导航
├── components/
│   ├── TeacherCheckInTab.vue      # 教师打卡功能组件
│   ├── StudentAttendanceTab.vue   # 学生考勤管理组件
│   ├── AttendanceStatisticsTab.vue # 考勤统计分析组件
│   ├── AttendanceHistoryTab.vue   # 考勤历史记录组件
│   └── README.md                  # 功能说明文档
└── README.md                      # 本文档
```

## 功能特性

### 1. 教师打卡功能 (TeacherCheckInTab.vue)

**主要功能：**
- 实时显示当前时间
- 今日打卡状态展示（签到、签退、工作时长）
- 快速签到/签退按钮
- 请假申请（病假、事假、年假、产假）
- 本月考勤日历视图
- 月度考勤统计

**特色功能：**
- 自动迟到/早退检测
- 实时工作时长计算
- 考勤日历可视化
- 请假流程集成

### 2. 学生考勤管理 (StudentAttendanceTab.vue)

**主要功能：**
- 班级和学生筛选
- 实时考勤统计展示（总人数、出勤、缺勤、出勤率）
- 学生考勤录入（支持批量操作）
- 健康状态监测（体温、健康状况）
- 考勤详情编辑
- 滑动操作支持

**特色功能：**
- 一键批量签到
- 体温异常预警
- 学生头像显示
- 滑动编辑操作

### 3. 考勤统计分析 (AttendanceStatisticsTab.vue)

**主要功能：**
- 教师个人考勤统计
- 班级学生考勤分析
- 出勤趋势图表展示
- 学生出勤排行榜
- 异常学生识别和提醒

**统计维度：**
- 出勤天数、迟到次数、早退次数
- 出勤率、平均出勤人数
- 异常次数统计
- 考勤分布可视化

### 4. 考勤历史记录 (AttendanceHistoryTab.vue)

**主要功能：**
- 历史记录查询（教师/学生）
- 多维度筛选（时间、状态、班级、学生）
- 记录详情展示
- 数据导出功能
- 记录分享功能

**筛选选项：**
- 记录类型（教师/学生）
- 时间范围选择
- 考勤状态筛选
- 班级/学生筛选

## 使用指南

### 教师打卡操作流程

1. **签到操作**
   - 打开教师考勤页面
   - 点击"教师打卡"标签
   - 查看今日打卡状态
   - 点击"签到"按钮完成签到

2. **签退操作**
   - 在完成当日工作后
   - 点击"签退"按钮完成签退
   - 系统自动计算工作时长

3. **请假申请**
   - 点击"请假申请"按钮
   - 选择请假类型和时间
   - 填写请假原因
   - 提交申请等待审批

### 学生考勤管理流程

1. **选择班级**
   - 切换到"学生考勤"标签
   - 选择要管理的班级
   - 查看班级学生列表

2. **批量签到**
   - 点击"批量签到"按钮
   - 确认操作完成全班签到

3. **个别考勤编辑**
   - 滑动学生条目
   - 点击"编辑"按钮
   - 修改考勤状态和相关信息
   - 保存更改

4. **保存考勤记录**
   - 完成所有编辑后
   - 点击"保存考勤"按钮
   - 系统验证并保存数据

### 统计分析使用

1. **查看个人统计**
   - 选择"教师个人"统计类型
   - 查看个人出勤统计数据
   - 分析出勤趋势和分布

2. **查看班级统计**
   - 选择"班级学生"统计类型
   - 选择要分析的班级
   - 查看班级整体考勤情况

### 历史记录查询

1. **设置筛选条件**
   - 选择记录类型
   - 设置时间范围
   - 选择状态筛选条件

2. **查看记录详情**
   - 点击具体记录
   - 查看完整的考勤信息
   - 支持分享功能

## 权限控制

- **教师权限**：只能管理自己负责的班级
- **时间限制**：只能修改当天的考勤记录
- **数据隔离**：不同用户看到不同的数据范围

## 注意事项

1. **网络要求**：需要稳定的网络连接进行数据同步
2. **时间同步**：确保设备时间准确，影响考勤计算
3. **权限验证**：需要正确的用户权限才能访问相关功能
4. **数据备份**：重要操作后建议及时确认数据保存

## 故障排除

### 常见问题

1. **无法签到/签退**
   - 检查网络连接
   - 确认用户权限
   - 检查是否已经签到/签退

2. **班级列表为空**
   - 确认用户已分配班级
   - 检查权限设置
   - 联系管理员分配班级

3. **统计数据不准确**
   - 刷新页面重新加载
   - 检查时间筛选条件
   - 确认数据完整性

4. **导出功能失败**
   - 检查网络连接
   - 确认选择条件正确
   - 检查文件权限
