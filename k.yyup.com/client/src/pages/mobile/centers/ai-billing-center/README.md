# AI计费中心 - 移动端

## 页面概述

移动端AI计费中心页面严格按照PC端1:1复制原则开发，提供完整的AI用量监控和计费管理功能。

## 主要功能

### 1. 统计概览
- **总费用统计**: 显示当前周期的总费用
- **调用次数**: 统计AI服务总调用次数
- **Token使用量**: 文本模型的Token消耗统计
- **视频时长**: 视频生成服务的总时长统计

### 2. 费用趋势图表
- **多周期切换**: 支持月度、周度、日度趋势查看
- **双轴图表**: 同时显示费用变化和调用次数
- **响应式设计**: 自适应移动端屏幕尺寸

### 3. 计费类型分布
- **Token计费**: 文本模型的Token费用
- **时长计费**: 视频/音频时长费用
- **次数计费**: 按次计费的服务费用
- **字符计费**: 字符处理类服务费用
- **可视化展示**: 使用环形图展示各类费用占比

### 4. 账单明细
- **搜索功能**: 支持按模型名称搜索
- **分页加载**: 移动端友好的分页体验
- **详情展示**: 完整的计费记录信息
- **状态标识**: 清晰的计费状态标签

### 5. 导出功能
- **CSV导出**: 支持账单数据CSV格式导出
- **Excel导出**: 预留Excel格式导出功能
- **移动端下载**: 兼容移动端文件下载

## 技术特性

### UI组件
- **RoleBasedMobileLayout**: 基于角色的移动端布局
- **Vant 4组件库**: 现代化移动端UI组件
- **响应式设计**: 适配各种移动设备屏幕

### 数据可视化
- **ECharts图表**: 专业的数据可视化方案
- **移动端优化**: 图表针对触摸交互优化
- **动态更新**: 实时数据刷新

### API集成
- **完整API**: 集成所有PC端计费相关API
- **错误处理**: 完善的错误提示和重试机制
- **数据缓存**: 优化加载性能
