## 检测方法

使用MCP浏览器工具进行检测：
1. 使用 Task 工具开启子代理
2. 子代理使用 MCP browser_navigate 导航到页面
3. 使用 browser_wait_for 等待页面加载
4. 使用 browser_snapshot 获取页面快照
5. 使用 browser_click 点击交互元素
6. 使用 browser_console_messages 获取控制台错误
7. 记录检测结果，更新状态
