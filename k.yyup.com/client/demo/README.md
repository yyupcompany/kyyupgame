# Demo - 现代化暗黑主题管理界面

这是一个使用Vue 3 + TypeScript + SCSS构建的现代化暗黑主题管理界面演示。

## 特性

- 🌙 **暗黑主题**: 精心设计的暗黑色彩方案，护眼且现代
- 🎨 **现代化设计**: 采用最新的UI设计趋势，色彩鲜艳，质感丰富
- 📱 **响应式布局**: 完美适配桌面端和移动端
- ⚡ **高性能**: 使用Vue 3 Composition API，性能优异
- 🔧 **TypeScript**: 完整的类型支持，开发体验更佳
- 🎭 **动画效果**: 流畅的过渡动画和交互反馈

## 目录结构

```
demo/
├── styles/           # 样式文件
│   └── global.scss   # 全局SCSS样式
├── layouts/          # 布局组件
│   ├── BaseLayout.vue    # 基础布局
│   ├── SideLayout.vue    # 侧边栏布局
│   └── TabbarLayout.vue  # 顶部导航布局
├── views/            # 页面组件
│   └── Dashboard.vue # 仪表板页面
├── components/       # 通用组件
├── router.ts         # 路由配置
├── main.ts          # 应用入口
├── index.html       # HTML模板
└── README.md        # 说明文档
```

## 组件说明

### BaseLayout.vue
- 主要的布局容器
- 管理侧边栏的展开/收起状态
- 响应式设计，移动端自动适配
- 包含页面切换动画

### SideLayout.vue
- 侧边栏导航组件
- 支持菜单项的激活状态
- 可折叠设计，节省空间
- 包含Logo区域和底部操作区

### TabbarLayout.vue
- 顶部导航栏组件
- 包含面包屑导航
- 搜索功能
- 通知中心
- 用户菜单
- 全屏切换

### Dashboard.vue
- 仪表板页面
- 统计卡片展示
- 图表数据可视化
- 活动时间线
- 快速操作面板

## 样式系统

### CSS变量
使用CSS自定义属性定义设计系统：

- **颜色**: 主色调、辅助色、状态色等
- **间距**: 统一的间距系统
- **字体**: 字号、行高、字重
- **圆角**: 统一的圆角规范
- **阴影**: 层次化的阴影系统
- **动画**: 统一的过渡时间

### 工具类
提供常用的CSS工具类：

- 布局类: `.flex`, `.flex-col`, `.items-center` 等
- 尺寸类: `.w-full`, `.h-full` 等
- 文本类: `.text-center`, `.cursor-pointer` 等

### 组件样式
- 按钮样式: `.btn`, `.btn-primary`, `.btn-secondary` 等
- 卡片样式: `.card`
- 输入框样式: `.input`

## 响应式设计

- **桌面端**: 完整的侧边栏和顶部导航
- **平板端**: 自动收起侧边栏
- **移动端**: 侧边栏变为抽屉式，添加遮罩层

## 动画效果

- **页面切换**: 滑动进入/退出动画
- **菜单展开**: 淡入淡出效果
- **悬停反馈**: 微妙的变换和阴影变化
- **加载状态**: 旋转加载动画

## 使用方法

1. 确保已安装Vue 3和相关依赖
2. 在浏览器中打开 `index.html`
3. 或者配置开发服务器指向demo目录

## 自定义主题

可以通过修改 `global.scss` 中的CSS变量来自定义主题：

```scss
:root {
  --primary-color: #your-color;
  --bg-primary: #your-bg-color;
  // ... 其他变量
}
```
