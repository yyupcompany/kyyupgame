## 🎯 功能概述

教师客户跟踪SOP系统是一个专为幼儿园教师设计的客户关系管理工具，帮助教师系统化地跟踪潜在客户（家长），通过标准化的SOP（标准操作流程）提升招生转化率。

### 核心价值
- 📊 **标准化流程** - 7个标准化跟进阶段，确保每个客户都得到系统化跟进
- 🎯 **智能预测** - AI算法预测客户成功转化概率
- 💬 **对话管理** - 完整记录每次沟通内容和客户反馈
- 📈 **数据可视化** - 实时展示跟进进度和转化漏斗
- 🤖 **AI辅助** - 智能建议下一步行动和话术模板

#### 2.4 SOP阶段流程

**7个标准阶段**:

##### 阶段1: 初次接触
- **目标**: 建立初步联系，获取基本信息
- **任务清单**:
  - [ ] 自我介绍和幼儿园介绍
  - [ ] 了解客户基本需求
  - [ ] 获取联系方式
  - [ ] 添加微信好友
  - [ ] 发送幼儿园资料
- **话术模板**:
  ```
  您好，我是XX幼儿园的XX老师。
  很高兴认识您！请问您家宝宝多大了？
  我们幼儿园有适合X岁孩子的课程...
  ```
- **FAQ**:
  - Q: 幼儿园在哪里？
  - Q: 学费多少？
  - Q: 有哪些班级？

##### 阶段2: 需求了解
- **目标**: 深入了解客户需求和期望
- **任务清单**:
  - [ ] 了解孩子性格特点
  - [ ] 了解家长教育理念
  - [ ] 了解预算范围
  - [ ] 了解入园时间
  - [ ] 了解特殊需求
- **话术模板**:
  ```
  为了给您推荐最合适的班级，
  我想了解一下宝宝的情况...
  ```

##### 阶段3: 方案介绍
- **目标**: 提供定制化解决方案
- **任务清单**:
  - [ ] 准备个性化方案
  - [ ] 介绍课程体系
  - [ ] 介绍师资力量
  - [ ] 介绍园所环境
  - [ ] 发送方案资料
- **话术模板**:
  ```
  根据您的需求，我为您推荐...
  这个班级的特色是...
  ```

##### 阶段4: 试听体验
- **目标**: 邀请客户实地参观和试听
- **任务清单**:
  - [ ] 预约参观时间
  - [ ] 安排试听课程
  - [ ] 准备参观路线
  - [ ] 介绍园所设施
  - [ ] 收集体验反馈
- **话术模板**:
  ```
  欢迎您带宝宝来园参观！
  我们安排了XX课程的试听...
  ```

##### 阶段5: 价格谈判
- **目标**: 讨论价格和优惠政策
- **任务清单**:
  - [ ] 介绍收费标准
  - [ ] 说明优惠政策
  - [ ] 对比性价比
  - [ ] 处理价格异议
  - [ ] 提供分期方案
- **话术模板**:
  ```
  我们的收费标准是...
  现在报名有XX优惠...
  ```

##### 阶段6: 合同签订
- **目标**: 完成合同签订和缴费
- **任务清单**:
  - [ ] 准备合同文件
  - [ ] 讲解合同条款
  - [ ] 办理缴费手续
  - [ ] 收集入园资料
  - [ ] 建立学籍档案
- **话术模板**:
  ```
  恭喜您！我们来办理入园手续...
  请您准备以下资料...
  ```

##### 阶段7: 入园准备
- **目标**: 协助家长完成入园准备
- **任务清单**:
  - [ ] 发送入园须知
  - [ ] 安排家长会
  - [ ] 介绍班级老师
  - [ ] 准备入园物品清单
  - [ ] 安排适应期计划
- **话术模板**:
  ```
  为了让宝宝更好地适应幼儿园，
  我们准备了入园指南...
  ```

## 📊 数据模型

### 客户信息表 (customers)

```typescript
interface Customer {
  id: number;                    // 客户ID
  name: string;                  // 客户姓名
  phone: string;                 // 联系电话
  email?: string;                // 邮箱
  childName: string;             // 孩子姓名
  childAge: number;              // 孩子年龄
  childBirthday: Date;           // 孩子生日
  source: string;                // 客户来源
  address?: string;              // 居住地址
  budget?: string;               // 预算范围
  intendedClass?: string;        // 意向班级
  currentStage: number;          // 当前SOP阶段 (1-7)
  successProbability: number;    // 成功概率 (0-100)
  status: string;                // 状态 (跟进中/已转化/已流失)
  assignedTeacherId: number;     // 分配教师ID
  tags: string[];                // 客户标签
  remark?: string;               // 备注
  createdAt: Date;               // 创建时间
  updatedAt: Date;               // 更新时间
  lastFollowUpAt?: Date;         // 最后跟进时间
}
```

### SOP进度表 (sop_progress)

```typescript
interface SOPProgress {
  id: number;                    // 进度ID
  customerId: number;            // 客户ID
  stage: number;                 // 阶段 (1-7)
  stageName: string;             // 阶段名称
  progress: number;              // 阶段进度 (0-100)
  startTime: Date;               // 开始时间
  endTime?: Date;                // 完成时间
  duration: number;              // 停留时长(天)
  tasks: Task[];                 // 任务列表
  isCompleted: boolean;          // 是否完成
  createdAt: Date;
  updatedAt: Date;
}
```

### 对话记录表 (conversations)

```typescript
interface Conversation {
  id: number;                    // 记录ID
  customerId: number;            // 客户ID
  teacherId: number;             // 教师ID
  type: string;                  // 沟通方式 (电话/微信/面谈/短信)

  sentiment: number;             // 情感分析分数 (-1 to 1)
  attachments: Attachment[];     // 附件列表
  duration?: number;             // 沟通时长(分钟)
  createdAt: Date;               // 沟通时间
}
```

### 任务表 (tasks)

interface Task {

1. **安装依赖**
```bash
# 安装所有依赖
npm run install:all
```

2. **配置数据库**
```bash
# 配置环境变量
cp server/.env.example server/.env

# 编辑数据库配置
DB_HOST=dbconn.sealoshzh.site
DB_PORT=43906
DB_USER=root
DB_PASSWORD=pwk5ls7j
DB_NAME=kargerdensales
```

3. **初始化权限**
```bash
# 添加客户跟踪权限
cd server && node add-customer-tracking-permissions.cjs

# 添加SOP详情页权限
cd server && node add-sop-detail-permission.cjs
```

4. **启动服务**
```bash
# 同时启动前后端
npm run start:all

# 或分别启动
npm run start:frontend  # 前端 (端口5173)
npm run start:backend   # 后端 (端口3000)
```

### 访问地址

- **前端**: http://localhost:5173
- **后端API**: http://localhost:3000
- **API文档**: http://localhost:3000/api-docs

## 📝 使用指南

### 教师操作流程

#### 1. 创建新客户
1. 进入客户跟踪列表页
2. 点击"创建新客户"按钮
3. 填写客户基本信息
4. 保存并开始跟进

#### 2. 记录跟进
1. 进入客户SOP详情页
2. 在对话记录区域点击"添加记录"
3. 选择沟通方式
4. 填写沟通内容和客户反馈
5. 上传相关截图或附件
6. 保存记录

#### 3. 推进阶段
1. 完成当前阶段所有任务
2. 点击"推进到下一阶段"按钮
3. 系统自动更新进度
4. 开始下一阶段的跟进

#### 4. 查看AI建议
1. 在SOP详情页右侧查看AI建议面板
2. 参考AI推荐的下一步行动
3. 使用AI生成的话术模板
4. 注意AI提示的风险点
