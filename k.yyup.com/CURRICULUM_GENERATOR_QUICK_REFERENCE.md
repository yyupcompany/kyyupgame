## 💻 新增方法

### fillExample(example: string)
```typescript
// 点击示例提示时调用
function fillExample(example: string) {
  prompt.value = example;
}
```

### clearForm()
```typescript
// 清空表单
function clearForm() {
  prompt.value = '';
  selectedDomain.value = 'science';
  ageGroup.value = '4-5岁';
}
```
