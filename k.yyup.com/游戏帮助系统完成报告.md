## 🎨 设计规范

### 帮助按钮设计

```vue
<el-button circle @click="showHelp = true" type="info">
  <el-icon><QuestionFilled /></el-icon>
</el-button>
```

**特点**：
- 图标：❓ QuestionFilled
- 类型：info（蓝色）
- 形状：circle（圆形）
- 位置：顶部左侧，设置按钮右侧

### 帮助弹窗结构

```
🎮 游戏说明
├── 游戏标题（带emoji）
├── 游戏简介（灰色背景框）
├── 📖 游戏规则（4条有序列表）
├── 🎯 游戏目标（3条无序列表）
├── 📈 难度递增（分关卡说明）
├── 🎮 特殊功能（游戏独特机制）
└── 💡 游戏技巧（4条，彩色背景框）
```

### 样式特点

1. **主题色一致**：帮助说明的主题色与游戏主题色一致
2. **清晰层次**：h2/h3标题，有序/无序列表
3. **重点突出**：
   - 重要提示用橙色边框
   - 游戏技巧用彩色背景框
   - Strong标签用主题色高亮
4. **易读性**：
   - 合理的行高(1.6)
   - 清晰的间距
   - 舒适的字号

## 📖 帮助内容示例

### 水果记忆大师

```
游戏规则：
1. 游戏会按顺序点亮水果，请仔细观察并记住顺序
2. 演示完成后，按照刚才看到的顺序点击水果
3. 点击正确会获得分数，点击错误会失去生命值
4. 失去3次生命值后游戏结束

游戏目标：
- 记住水果的点亮顺序
- 按正确顺序重复点击
- 挑战更高关卡，测试记忆极限

难度递增：
- 第1关: 3个水果，序列长度3
- 第2关: 4个水果，序列长度4
- 第3关: 5个水果，序列长度5
- 第7关+: 9个水果，序列长度9+

游戏技巧：
- 集中注意力观察演示
- 可以小声重复水果名称帮助记忆
- 序列长了可以分段记忆
- 不要着急，想清楚再点击
```
