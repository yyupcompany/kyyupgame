# Claude Code Flow 中文汉化版使用指南

## 🎯 汉化版特性

### ✅ 已汉化内容
- **系统状态显示**: 状态、代理数量、任务队列等
- **SPARC模式描述**: 17种开发模式的中文说明
- **内存管理消息**: 初始化、存储、关闭等状态
- **错误和提示信息**: 常用的系统反馈
- **中文命令支持**: 支持中文命令输入

### 🌟 中文命令对照表

| 中文命令 | 英文命令 | 功能说明 |
|---------|---------|---------|
| `cf 状态` | `cf status` | 查看系统状态 |
| `cf 内存 列表` | `cf memory list` | 查看内存条目 |
| `cf 内存 存储` | `cf memory store` | 存储数据到内存 |
| `cf 内存 获取` | `cf memory get` | 从内存获取数据 |
| `cf 内存 统计` | `cf memory stats` | 内存使用统计 |
| `cf sparc 模式` | `cf sparc modes` | 列出SPARC模式 |
| `cf sparc 运行` | `cf sparc run` | 运行指定模式 |
| `cf 代理 列表` | `cf agent list` | 列出活跃代理 |
| `cf 代理 生成` | `cf agent spawn` | 创建新代理 |
| `cf 帮助` | `cf help` | 显示帮助信息 |
| `cf 开始` / `cf 启动` | `cf start` | 启动系统 |
| `cf 配置` | `cf config` | 配置管理 |
| `cf 监控` | `cf monitor` | 实时监控 |
| `cf 群体` / `cf 蜂群` | `cf swarm` | 群体协调 |

## 🚀 快速开始

### 1. 安装中文版脚本
```bash
# 复制中文汉化版脚本
cp /home/devbox/project/cf-chinese /usr/local/bin/cf
chmod +x /usr/local/bin/cf

# 或者创建别名
alias cf="/home/devbox/project/cf-chinese"
```

### 2. 基本使用
```bash
# 查看系统状态 (中文显示)
cf 状态

# 查看所有SPARC开发模式 (中文描述)
cf sparc 模式

# 查看内存中的数据
cf 内存 列表

# 直接输入中文任务描述 (自动启用多代理)
cf "检查我的Vue项目代码质量并提供优化建议"
cf "重构React组件使其更加模块化"
cf "优化我的数据库查询性能"
```

## 📋 SPARC模式中文说明

### 🎯 核心编排模式 (4种)
- **编排器**: 多代理任务编排和协调
- **群体协调器**: 高级群体管理和协调
- **工作流管理器**: 流程自动化和工作流编排
- **批处理执行器**: 并行任务执行和批量操作

### 🔧 开发模式 (4种)
- **编码器**: 自主代码生成，支持批量文件操作
- **架构师**: 基于内存的系统设计和架构规划
- **审查员**: 批量代码审查和质量检测
- **TDD**: 测试驱动开发，配合TodoWrite规划

### 📊 分析与研究模式 (3种)
- **研究员**: 并行深度研究，使用WebSearch/WebFetch
- **分析师**: 代码和数据分析，支持批量处理
- **优化器**: 性能优化，系统化分析方法

### 🎨 创意与支持模式 (4种)
- **设计师**: UI/UX设计，配合内存协调
- **创新者**: 创意问题解决，使用WebSearch
- **文档员**: 批量文档编写和文件操作
- **调试器**: 系统化调试，使用TodoWrite/Memory

### 🧪 测试与质量模式 (2种)
- **测试员**: 综合测试，支持并行执行
- **内存管理器**: 知识管理，使用Memory工具

## 🎯 使用技巧

### 1. 任务描述要具体
```bash

# ✅ 具体明确
cf "优化Vue组件的渲染性能，减少不必要的重渲染，使用Vue.memo和计算属性"
```

### 2. 善用SPARC模式
```bash

cf sparc 运行 TDD "用户认证功能"
cf sparc 运行 优化器 "数据库查询性能"
```

### 3. 利用内存协调
```bash

```

## 🐛 故障排除

### 常见问题
1. **语言编码警告**: `setlocale: LC_ALL: cannot change locale (zh_CN.UTF-8)`
   - 这是正常的，不影响功能使用
   - 可以忽略或安装中文语言包解决

2. **命令超时**:
   - 复杂任务可能需要更长时间
   - 可以使用 `--timeout` 参数调整

3. **代理启动失败**:
   - 检查网络连接
   - 确认Claude API配置正确

### 获取详细日志
```bash
# 启用详细输出
cf "任务描述" --verbose

# 查看系统状态
cf 状态

# 监控实时状态
cf 监控
```
