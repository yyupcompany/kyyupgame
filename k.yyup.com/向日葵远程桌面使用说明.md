## 🚀 如何使用

### 1. 获取连接信息
在桌面环境中:
1. 查看系统托盘区域的向日葵图标
2. 左键点击图标显示**连接码**
3. 右键点击图标 → 设置 → 查看或修改**连接密码**

### 2. 远程连接此电脑

#### 从其他设备连接:
**手机/平板:**
1. 下载"向日葵远程控制"App
2. 输入连接码和密码
3. 开始远程控制

**电脑:**
1. 方法A: 访问 https://sunlogin.oray.com/
   - 输入连接码
   - 选择网页版或下载客户端
2. 方法B: 下载向日葵客户端
   - 安装后输入连接码和密码

### 3. 连接码获取命令
```bash
# 查看向日葵状态
./sunlogin-manager.sh status

# 显示连接帮助
./sunlogin-manager.sh info
```

## 🔧 管理命令

### 向日葵控制脚本
```bash
# 查看帮助
./sunlogin-manager.sh help

# 启动服务
./sunlogin-manager.sh start

# 停止服务
./sunlogin-manager.sh stop

# 重启服务
./sunlogin-manager.sh restart

# 查看状态
./sunlogin-manager.sh status
```

## 🔒 安全设置

### 建议配置
1. **修改默认密码**:
   - 右键向日葵图标 → 设置 → 安全设置
   - 设置复杂密码

2. **启用访问验证**:
   - 开启"需要验证"选项
   - 设置临时访问权限

3. **连接日志**:
   - 查看连接历史
   - 设置异常提醒

## 🌐 网络要求

### 防火墙设置
如连接失败，请确保:
```bash
# 允许向日葵端口
sudo ufw allow 3389/tcp
sudo ufw allow 5900/tcp
```

### 网络状态
- ✅ 局域网连接: 直接使用IP地址
- ✅ 互联网连接: 使用连接码
- ✅ 穿透NAT: 自动处理

## 🆘 故障排除

### 常见问题
1. **连接码获取失败**
   ```bash
   # 重启向日葵
   ./sunlogin-manager.sh restart
   ```

2. **无法连接**
   - 检查网络连接
   - 确认防火墙设置
   - 验证连接码和密码

3. **显示问题**
   - 安装桌面环境依赖
   - 更新显卡驱动

### 手动启动
```bash
# 如果自动启动失败
/opt/apps/com.oray.sunlogin.client/files/bin/sunloginclient &
```

## 📋 文件位置

- **管理脚本**: `./sunlogin-manager.sh`
