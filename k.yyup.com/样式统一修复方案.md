## 🎯 统一样式规范

### 1. 容器类名规范

**统一使用**: `center-container`

```vue
<template>
  <div class="center-container">
    <!-- 页面内容 -->
  </div>
</template>
```

**样式特性**:
- 最小高度: 100vh
- 背景色: var(--bg-secondary, #f5f7fa)
- 内边距: 20px
- 玻璃态效果
- 响应式适配

---

### 2. 页面头部规范

**统一使用**: `center-page-header` 或 `page-header`

```vue
<div class="page-header">
  <div class="header-left">
    <h1 class="page-title">页面标题</h1>
    <p class="page-subtitle">页面描述</p>
  </div>
  <div class="header-actions">
    <el-button type="primary">操作按钮</el-button>
  </div>
</div>
```

**样式特性**:
- 渐变背景: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
- 圆角: 16px
- 阴影: 0 8px 24px rgba(102, 126, 234, 0.3)
- 白色文字
- 响应式布局

---

### 3. 卡片样式规范

**统一使用**: `center-card` 或 Element Plus 的 `el-card`

```vue
<div class="center-card">
  <div class="card-header">
    <h3 class="card-title">卡片标题</h3>
    <div class="card-actions">操作按钮</div>
  </div>
  <div class="card-content">
    <!-- 卡片内容 -->
  </div>
</div>
```

**样式特性**:
- 背景: 白色
- 圆角: 12px
- 阴影: 0 4px 16px rgba(0, 0, 0, 0.08)
- 边框: 1px solid #e5e7eb
- Hover效果: 上移2px + 阴影加深

---

### 4. 统计卡片规范

**统一使用**: `stats-grid` + `stat-card`

```vue
<div class="stats-grid">
  <div class="stat-card">
    <div class="stat-content">
      <div class="stat-icon">
        <el-icon><User /></el-icon>
      </div>
      <div class="stat-info">
        <div class="stat-value">1,234</div>
        <div class="stat-label">标签</div>
      </div>
    </div>
  </div>
</div>
```

**样式特性**:
- 网格布局: repeat(auto-fit, minmax(240px, 1fr))
- 间距: 20px
- 响应式: 移动端单列
- Hover效果: 上移4px

---

### 5. 表格样式规范

**统一使用**: Element Plus 表格 + 统一配置

```vue
<el-table
  :data="tableData"
  stripe
  border
  style="width: 100%"
>
  <el-table-column prop="id" label="ID" width="80" align="center" />
  <el-table-column prop="name" label="名称" width="200" show-overflow-tooltip />
  <!-- 更多列 -->
</el-table>
```

**配置规范**:
- 使用 `stripe` 斑马纹
- 使用 `border` 边框
- 固定宽度使用 `width`，不使用 `min-width`
- 长文本使用 `show-overflow-tooltip`
- 居中对齐使用 `align="center"`
