## 🔄 SOP标准流程（7个阶段）

### 阶段1: 初次接触（1天）

**目标**: 建立第一印象，获取基本信息

**核心任务**:
- ✅ 自我介绍（5分钟）
- ✅ 了解基本信息（10分钟）
- ✅ 建立信任（10分钟）

**成功标志**:
- 客户愿意继续沟通
- 约定了下次联系时间
- 获得了基本需求信息

**AI建议**: 个性化开场白、破冰话术

---

### 阶段2: 需求挖掘（2天）

**目标**: 深入了解客户需求和痛点

**核心任务**:
- ✅ 了解孩子情况
- ✅ 挖掘教育需求
- ✅ 识别痛点
- ✅ 确认预算范围

**成功标志**:
- 明确了客户的核心需求
- 找到了客户的痛点
- 了解了决策流程

**AI建议**: 提问清单、需求分析

---

### 阶段3: 方案呈现（2天）

**目标**: 展示课程价值，匹配客户需求

**核心任务**:
- ✅ 发送课程介绍
- ✅ 邀请试听课
- ✅ 发送成功案例
- ✅ 解答疑问

**成功标志**:
- 客户参加了试听课
- 客户认可课程价值
- 客户表现出购买意向

**AI建议**: 方案定制、案例推荐

---

### 阶段4: 异议处理（1-2天）

**目标**: 解决客户顾虑，消除购买障碍

**核心任务**:
- ✅ 识别异议类型
- ✅ 针对性解答
- ✅ 提供证明材料
- ✅ 建立紧迫感

**成功标志**:
- 主要异议已解决
- 客户进入决策阶段
- 明确了购买时间表

**AI建议**: 异议分析、应对话术

---

### 阶段5: 临门一脚（1天）

**目标**: 促成最终决策

**核心任务**:
- ✅ 确认购买意向
- ✅ 提供优惠方案
- ✅ 设定截止时间
- ✅ 协助决策

**成功标志**:
- 客户明确表示要购买
- 约定了签约时间
- 准备好了相关材料

**AI建议**: 临门话术、优惠策略

---

### 阶段6: 成交签约（1天）

**目标**: 完成签约流程

**核心任务**:
- ✅ 准备合同
- ✅ 讲解条款
- ✅ 完成签约
- ✅ 收取费用

**成功标志**:
- 合同已签署
- 费用已收取
- 客户满意度高

**AI建议**: 签约注意事项

---

### 阶段7: 售后服务（持续）

**目标**: 确保客户满意，获取转介绍

**核心任务**:
- ✅ 入学引导
- ✅ 定期回访
- ✅ 收集反馈
- ✅ 请求转介绍

**成功标志**:
- 客户满意度高
- 获得了转介绍
- 建立了长期关系

**AI建议**: 服务话术、转介绍技巧

## 🤖 AI智能建议

### 触发方式

1. **任务级建议**
   - 每个SOP任务旁边有"AI建议"按钮
   - 点击即可获取针对性建议

2. **全局分析**
   - 页面顶部"AI全局分析"按钮
   - 提供客户的全面分析

3. **自动建议**
   - 完成任务后自动提示下一步
   - 上传截图后自动分析

### AI建议内容

#### 1. 沟通策略
```
基于客户画像和历史记录，提供：
- 沟通方式建议（温和/直接/专业等）
- 沟通重点（强调什么）
- 注意事项（避免什么）
```

#### 2. 推荐话术
```
提供3-5条可直接使用的话术：
- 开场白
- 核心话术
- 结束语
- 异议处理话术
```

#### 3. 下一步行动
```
明确告诉教师：
- 什么时候做（时机）
- 做什么（具体动作）
- 怎么做（执行步骤）
- 为什么（原因和目的）
```

#### 4. 成功概率预测
```
基于数据分析：
- 成交概率（百分比）
- 影响因素（正面/负面）
- 改进建议（如何提高概率）
```

### AI调用流程

```typescript
// 1. 收集上下文
const context = {
  customer: customerInfo,           // 客户信息
  conversations: conversationHistory, // 对话历史
  followRecords: followRecords,     // 跟进记录
  screenshots: screenshots,         // 截图
  currentStage: currentStage,       // 当前阶段
  currentTask: currentTask          // 当前任务
};

// 2. 调用AIBridge服务
const aiResponse = await aibridge.analyze({
  type: 'task-suggestion',
  context: context,
  prompt: buildPrompt(context)
});

// 3. 展示AI建议
showAISuggestion(aiResponse);
```
