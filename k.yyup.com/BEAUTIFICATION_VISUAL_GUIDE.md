## 🎯 交互效果

### 悬停效果

| 元素 | 效果 |
|------|------|
| 页面头部 | 上升 2px + 阴影增强 |
| 统计卡片 | 上升 4px + 阴影增强 + 边框变主色 |

| 标签页 | 下划线变主色 + 字体加粗 |

### 过渡动画

- **快速** (`--transition-fast`): 标签页、箭头
- **正常** (`--transition-normal`): 卡片、图表、表格行
- **缓慢** (`--transition-slow`): 页面加载

## ✨ 装饰性元素

所有主要容器都添加了径向渐变装饰:

```scss
&::before {
  content: '';
  position: absolute;
  background: radial-gradient(
    circle,
    var(--primary-light-bg) 0%,
    transparent 70%
  );
  opacity: 0.5;
  pointer-events: none;
}
```

这样可以:
- ✅ 增加视觉层次感
- ✅ 引导用户注意力
- ✅ 提升整体美感
- ✅ 不影响交互
