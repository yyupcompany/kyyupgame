## UI/UX 设计优化建议

### 1. 布局结构优化

- 确保使用统一的布局容器：`.app-container` > `.main-layout-area` > `.sidebar-slot` + `.content-slot`
- 保持 header 高度为 64px，使用 `--header-height` CSS 变量
- 侧边栏默认宽度 280px，收缩后 80px

### 2. 间距系统标准化

- 使用 CSS 变量统一管理间距：
  - `--spacing-xs`: 4px
  - `--spacing-sm`: 8px
  - `--spacing-md`: 16px
  - `--spacing-lg`: 24px
  - `--spacing-xl`: 32px

### 3. 颜色系统统一

- 使用 Element Plus 主题变量
- 自定义颜色通过 CSS 变量管理：
  - `--primary-color`: 主色
  - `--bg-color-page`: 页面背景
  - `--bg-card`: 卡片背景
  - `--text-primary`: 主要文字
  - `--border-color`: 边框颜色

### 4. 响应式断点

- `xs`: < 640px (移动端)
- `sm`: 640px - 767px
- `md`: 768px - 1023px (平板)
- `lg`: 1024px - 1279px
- `xl`: >= 1280px (桌面)

### 5. 组件样式一致性

- 按钮：高度 32px/40px，圆角 4px
- 卡片：圆角 8px，阴影 `var(--shadow-sm)`
- 表格：表头背景 `#f5f7fa`，斑马纹 `#fafafa`
- 弹窗：圆角 12px，最大宽度 50%
