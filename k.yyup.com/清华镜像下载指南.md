# 清华镜像站下载指南

## 下载地址（按推荐顺序）

### 1. Ubuntu Desktop (标准桌面版)
```
https://mirrors.tuna.tsinghua.edu.cn/ubuntu-releases/22.04.5/ubuntu-22.04.5-desktop-amd64.iso
```

### 2. Xubuntu (轻量级桌面版)
```
https://mirrors.tuna.tsinghua.edu.cn/xubuntu-releases/22.04/release/xubuntu-22.04.5-desktop-amd64.iso
```

### 3. Kubuntu (KDE桌面版)
```
https://mirrors.tuna.tsinghua.edu.cn/kubuntu-releases/22.04/release/kubuntu-22.04.5-desktop-amd64.iso
```

## 下载说明

1. 选择上述任一链接下载
2. 文件大小约2.5GB-4GB不等
3. 下载完成后不要解压
4. 建议使用下载工具（如迅雷）下载

## 版本说明

1. **Ubuntu Desktop**：
   - 最常用的版本
   - GNOME桌面环境
   - 界面美观，功能完整

2. **Xubuntu**：
   - 轻量级桌面环境
   - 占用资源少
   - 适合配置较低的电脑

3. **Kubuntu**：
   - KDE桌面环境
   - 界面华丽
   - 类似Windows的操作方式

## 校验文件完整性

下载完成后可以通过以下命令验证文件完整性：
```powershell
certutil -hashfile 下载的文件路径 SHA256
```
