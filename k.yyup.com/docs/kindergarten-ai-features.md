# 🎨 幼儿园AI智能配图系统

## 📋 功能概述

专为3-6岁幼儿园场景设计的AI图片生成系统，提供温馨、安全、富有童趣的图片内容，满足幼儿园日常教学、宣传、管理等多种需求。

## ✨ 核心特色

### 🎯 专业幼教场景
- **室内教室环境**：温馨的学习空间，适合各种教学活动
- **户外操场环境**：安全的游戏设施，充满活力的运动场景
- **餐厅用餐环境**：营养丰富的用餐时光，培养良好习惯
- **午休室环境**：安静舒适的休息空间
- **表演舞台环境**：展示才艺的舞台，培养自信
- **手工制作环境**：创意无限的艺术空间

### 🎨 多样化风格
- **卡通可爱风格**：最受3-6岁孩子喜爱，色彩鲜艳温馨
- **自然温馨风格**：真实自然的场景，温暖的家庭氛围
- **真实摄影风格**：专业的摄影效果，展现真实美好

### 👶 精准年龄定位
- **小班 (3-4岁)**：适合小班幼儿的活动和环境
- **中班 (4-5岁)**：符合中班孩子发展特点的场景
- **大班 (5-6岁)**：适应大班幼儿能力的活动内容
- **混龄 (3-6岁)**：适合不同年龄段共同参与的活动

## 🛠️ 技术实现

### 后端服务
- **自动配图服务**：`AutoImageGenerationService`
- **专业提示词优化**：针对幼儿园场景的关键词增强
- **安全内容过滤**：确保生成内容适合幼儿观看
- **多模型支持**：支持不同AI图像生成模型

### 前端组件
- **通用配图组件**：`AutoImageGenerator.vue`
- **幼儿园专用组件**：`KindergartenImageGenerator.vue`
- **演示页面**：`KindergartenAIDemo.vue`

### API接口
```typescript
// 生成图片
POST /api/auto-image/generate
{
  "prompt": "3-6岁的小朋友们在温馨的幼儿园教室里快乐地学习",
  "category": "activity",
  "style": "cartoon",
  "size": "1024x768",
  "quality": "standard"
}

// 检查服务状态
GET /api/auto-image/status

// 获取生成历史
GET /api/auto-image/history
```

## 📱 使用方式

### 1. 快速生成
点击预设模板按钮，一键生成对应场景的图片：
```vue
<KindergartenImageGenerator
  :auto-use="false"
  @image-generated="handleImageGenerated"
  @image-used="handleImageUsed"
/>
```

### 2. 自定义生成
通过表单自定义生成参数：
- 活动描述：详细描述想要的场景
- 年龄段：选择适合的年龄组
- 场景类型：选择室内外环境
- 图片风格：选择视觉风格
- 图片尺寸：选择合适的尺寸

### 3. 集成使用
在活动模板、海报设计等页面中集成使用：
```vue
<template>
  <div class="template-actions">
    <el-button @click="useTemplate">使用模板</el-button>
    <KindergartenImageGenerator
      @image-used="updateTemplateImage"
    />
  </div>
</template>
```
