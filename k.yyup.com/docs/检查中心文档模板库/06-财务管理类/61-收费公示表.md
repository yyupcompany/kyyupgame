## 收费管理制度

### 一、收费原则

1. 依法收费，不得擅自设立收费项目、提高收费标准
2. 公开透明，主动公示收费项目和标准
3. 规范管理，使用合法票据
4. 专款专用，保教费主要用于保育教育活动

### 二、收费流程

1. 制定收费标准，报物价部门备案
2. 在显著位置公示收费项目和标准
3. 与家长签订入园协议，明确收费事项
4. 按月收取费用，当场开具票据
5. 建立收费台账，定期公布收支情况
6. 按规定办理退费手续

### 三、收费监督

1. 接受教育、物价、财政等部门监督检查
2. 接受家长和社会监督
3. 设立投诉电话，及时处理投诉
4. 定期开展收费自查，及时整改问题
