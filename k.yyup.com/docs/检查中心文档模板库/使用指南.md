## 📝 使用步骤

### 步骤1：选择模板

根据需要选择对应的模板文件，例如：
```
docs/检查中心文档模板库/02-专项检查类/13-幼儿晨检记录表.md
```

### 步骤2：复制模板

将模板文件复制到工作目录，重命名为实际使用的文件名，例如：
```
2025年10月9日-大一班-幼儿晨检记录表.md
```

### 步骤3：替换变量

使用文本编辑器打开文件，替换所有 `{{变量}}` 为实际内容：

**常用变量**:
- `{{幼儿园名称}}` → 阳光幼儿园
- `{{班级名称}}` → 大一班
- `{{检查日期}}` → 2025年10月9日
- `{{检查人员}}` → 张老师
- `{{幼儿姓名}}` → 小明
- `{{体温}}` → 36.5

**批量替换技巧**:
1. 使用文本编辑器的"查找替换"功能
2. 先替换通用变量（如幼儿园名称）
3. 再替换具体变量（如幼儿姓名）

### 步骤4：填写内容

根据实际情况填写表格内容：
- 删除不需要的行
- 添加需要的行
- 调整表格格式

### 步骤5：保存使用

- **电子存档**: 保存为PDF或Word格式
- **打印使用**: 直接打印使用
- **归档管理**: 按类别和日期归档

**使用脚本批量替换**:
```bash

sed -i 's/{{幼儿园名称}}/阳光幼儿园/g' *.md

# 批量替换日期
```

## 💡 常见问题

### Q1: 如何选择合适的模板？

**A**: 根据检查类型和使用频率选择：
- 年检：选择年度检查类模板
- 日常：选择专项检查类模板
- 督导：选择常态化督导类模板

### Q2: 模板可以修改吗？

**A**: 可以，建议：
- 保留原模板作为备份
- 根据实际需求调整
- 保持基本结构不变

### Q3: 如何批量生成多个文件？

**A**: 
- 使用脚本批量复制
- 使用Excel批量生成
- 使用模板管理系统

### Q4: 模板保存多久？

**A**: 根据文档类型：
- 年检材料：永久保存
- 日常记录：3年以上
- 临时记录：1年以上

### Q5: 如何确保数据安全？

**A**: 
- 定期备份
- 加密存储
- 权限管理
- 定期审计
