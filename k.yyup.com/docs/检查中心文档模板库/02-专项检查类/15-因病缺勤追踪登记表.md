## 因病缺勤追踪制度

### 一、追踪对象

1. 因病请假的幼儿
2. 疑似传染病的幼儿
3. 确诊传染病的幼儿
4. 传染病密切接触者

### 二、追踪频次

1. **普通疾病**: 每2-3天追踪一次
2. **传染病疑似**: 每日追踪
3. **传染病确诊**: 每日追踪，直至复课
4. **密切接触者**: 每日追踪，观察期内

### 三、追踪内容

1. 体温变化
2. 症状变化
3. 就医情况
4. 治疗情况
5. 饮食睡眠
6. 精神状态
7. 复课时间

### 四、追踪方式

1. 电话追踪
2. 微信追踪
3. 家访追踪
4. 要求家长主动报告

### 五、记录要求

1. 及时记录，不得漏记
2. 内容真实，详细准确
3. 分类管理，便于查阅
4. 妥善保存，至少3年

### 六、报告制度

1. 发现传染病疑似病例立即报告保健医生
2. 达到疫情报告标准立即报告疾控中心
3. 每日汇总因病缺勤情况报园长
4. 每周分析疾病流行趋势

## 填写说明

1. **及时追踪**: 幼儿因病缺勤当天即开始追踪
2. **详细记录**: 疾病名称、症状、就诊情况等详细记录
3. **每日更新**: 传染病病例每日追踪并更新记录
4. **分析总结**: 每月进行统计分析，掌握疾病流行趋势
5. **及时报告**: 达到疫情报告标准立即报告
6. **复课审核**: 严格按照复课标准审核，防止疫情扩散
7. **资料保存**: 追踪记录保存期限不少于3年
