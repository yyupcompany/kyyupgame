## 食品留样制度

### 一、留样目的

食品留样是为了在发生食物中毒或疑似食物中毒事故时，能够及时提供检验样品，查明事故原因，为事故处理提供依据。

### 二、留样要求

1. **留样范围**: 所有供应给幼儿的食品，包括主食、副食、汤类、水果等
2. **留样数量**: 每个品种不少于125克
3. **留样容器**: 使用专用密闭容器，每次使用前必须清洗消毒
4. **留样标识**: 标明食品名称、留样日期、留样时间、留样人
5. **留样温度**: 冷藏保存，温度控制在0-10℃
6. **留样时长**: 保存48小时以上
7. **留样记录**: 详细记录留样信息，包括食品名称、留样时间、留样人等

### 三、留样流程

1. **准备**: 清洗消毒留样容器，贴好标签
2. **留样**: 食品烹饪完成后，立即取样，不少于125克
3. **密封**: 盖好容器盖，确保密封
4. **冷藏**: 立即放入留样冰箱，温度0-10℃
5. **记录**: 填写留样记录表
6. **保存**: 保存48小时以上
7. **销毁**: 超过48小时后，确认无异常情况，方可销毁

### 四、留样管理

1. **专人负责**: 指定专人负责食品留样工作
2. **专柜存放**: 设置专用留样冰箱，不得存放其他物品
3. **定期检查**: 每天检查留样冰箱温度，确保在0-10℃
4. **及时销毁**: 超过48小时的留样食品，及时销毁，不得食用
5. **记录保存**: 留样记录保存期限不少于2年

### 五、应急处理

发生食物中毒或疑似食物中毒事故时：
1. 立即停止销毁留样食品
2. 封存留样食品，不得移动
3. 配合卫生部门进行调查
4. 提供留样食品供检验
