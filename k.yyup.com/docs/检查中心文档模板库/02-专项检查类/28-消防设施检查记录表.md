## 消防设施检查制度

### 一、检查频次

1. **日常检查**: 每日巡查，重点检查疏散通道、安全出口
2. **周检查**: 每周检查，检查灭火器、应急照明等
3. **月检查**: 每月检查，全面检查各类消防设施
4. **季度检查**: 每季度检查，测试报警系统、喷淋系统
5. **年度检查**: 每年检查，委托专业机构全面检测

### 二、检查要求

1. 按规定频次检查，不得遗漏
2. 逐项检查，认真记录
3. 发现问题，立即整改
4. 重大隐患，立即报告
5. 建立台账，妥善保存

### 三、维护保养

1. 定期维护保养，确保完好有效
2. 及时更换损坏设施
3. 委托专业机构年检
4. 建立维保档案
