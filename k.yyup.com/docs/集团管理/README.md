## 🔍 关键概念速查

### 核心表结构

| 表名 | 说明 | 关键字段 |
|------|------|----------|
| `groups` | 集团表 | id, name, code, investor_id |
| `group_users` | 集团用户关联表 | group_id, user_id, role |
| `kindergartens` | 幼儿园表 | id, group_id, is_group_headquarters |

### 集团角色

| 角色 | 代码 | 权限 |
|------|------|------|
| 投资人 | 1 | 最高权限 |
| 集团管理员 | 2 | 管理权限 |
| 财务总监 | 3 | 财务权限 |
| 运营总监 | 4 | 运营权限 |

### 核心API端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/groups` | GET | 获取集团列表 |
| `/api/groups/:id` | GET | 获取集团详情 |
| `/api/groups` | POST | 创建集团 |
| `/api/groups/upgrade` | POST | 升级为集团 |
| `/api/groups/:id/statistics` | GET | 获取集团统计 |
