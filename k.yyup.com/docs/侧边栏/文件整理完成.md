## 🔧 工具脚本

### 检查脚本

1. **check-chinese-names.js**
   - 功能：检查所有中心的中文名称
   - 使用：`cd docs/侧边栏 && node check-chinese-names.js`
   - 输出：显示所有活跃中心的配置信息

2. **check-disabled-centers.js**
   - 功能：检查已禁用中心的状态和权限关联
   - 使用：`cd docs/侧边栏 && node check-disabled-centers.js`
   - 输出：显示已禁用中心及其权限关联情况

### 清理脚本

3. **cleanup-disabled-centers.js**
   - 功能：清理已禁用中心的角色权限关联
   - 使用：`cd docs/侧边栏 && node cleanup-disabled-centers.js`
   - 作用：确保数据一致性，删除无效的权限关联

4. **clear-permission-cache.js**
   - 功能：清除Redis中的权限缓存
   - 使用：`cd docs/侧边栏 && node clear-permission-cache.js`
   - 作用：解决缓存导致的显示问题
