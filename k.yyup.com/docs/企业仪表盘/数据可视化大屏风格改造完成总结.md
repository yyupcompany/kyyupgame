## 🎨 核心设计元素

### 配色方案

| 元素 | 颜色 | 说明 |
|------|------|------|
| 主背景 | `#0a1929 → #1e3a5f` | 深蓝色渐变 |
| 卡片背景 | `rgba(15, 23, 42, 0.4-0.6)` | 半透明深色 |
| 主题色 | `#60a5fa` | 霓虹蓝色 |
| 边框色 | `rgba(59, 130, 246, 0.2-0.6)` | 半透明蓝色 |
| 标题文字 | `#fff` | 白色 + 发光 |
| 数值文字 | `#60a5fa` | 蓝色 + 发光 |
| 次要文字 | `#94a3b8` | 浅灰蓝 |
| 成功色 | `#22c55e` | 绿色 + 发光 |
| 警告色 | `#fb923c` | 橙色 + 发光 |
| 危险色 | `#ef4444` | 红色 + 发光 |

### 发光效果

#### 文字发光
```scss
text-shadow: 0 0 10px rgba(96, 165, 250, 0.5);
```

#### 边框发光
```scss
box-shadow: 
  0 0 20px rgba(59, 130, 246, 0.2),
  inset 0 0 20px rgba(59, 130, 246, 0.05);
```

#### 图标发光
```scss
filter: drop-shadow(0 0 10px rgba(96, 165, 250, 0.6));
```

### 毛玻璃效果
```scss
backdrop-filter: blur(10px);
```

### 等宽字体（数字）
```scss
font-family: 'Courier New', monospace;
```
