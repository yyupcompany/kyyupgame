## 🎨 视觉设计

### 配色方案
- **背景**: 深蓝色渐变 (#0a1929 → #1e3a5f)
- **卡片**: 半透明深色 (rgba(15, 23, 42, 0.6))
- **边框**: 霓虹蓝色 (rgba(59, 130, 246, 0.3))
- **文字**: 白色/蓝色发光效果
- **数字**: 等宽字体 (Courier New)

### 视觉效果
- ✅ 毛玻璃半透明效果 (`backdrop-filter: blur(10px)`)
- ✅ 霓虹蓝色发光边框
- ✅ 文字发光效果 (`text-shadow`)
- ✅ 悬停放大动画
- ✅ 渐变色图标背景

---

## 📐 布局尺寸

### 三列宽度
- **左侧**: 400px (固定宽度)
- **中间**: 1fr (自适应)
- **右侧**: 500px (固定宽度)

### 卡片尺寸
- **大卡片**: 自适应高度，3个平均分配
- **中卡片**: 紧凑布局，5个垂直排列
- **翻牌器**: 40px × 55px 每个数字

```typescript

  name: 'EnterpriseDashboardScreen',

  meta: {
    title: '企业数据大屏',

    requiresAuth: true,
    priority: 'high'
  }

```

- Vue 3 Composition API
- TypeScript
- SCSS (scoped)
- Element Plus Icons
- 实时时间更新

## 📊 数据结构

### 左侧大卡片数据
```typescript
{
  name: string        // 中心名称
  icon: Component     // 图标组件
  colorClass: string  // 颜色类名
  metrics: {
    label: string     // 指标名称
    value: string     // 指标值
  }[]
  trend: number       // 趋势百分比
}
```

### 右侧中卡片数据
```typescript
{
  name: string
  icon: Component
  colorClass: string
  metric1: { label: string, value: string }
  metric2: { label: string, value: string }
  trend: number
}
```
