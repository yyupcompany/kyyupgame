## 🗄️ 二、数据库设计

### 2.1 游戏配置表 (game_configs)
```sql
CREATE TABLE game_configs (
  id INT PRIMARY KEY AUTO_INCREMENT,
  game_key VARCHAR(50) UNIQUE NOT NULL,        -- 游戏标识（princess-garden）
  game_name VARCHAR(100) NOT NULL,             -- 游戏名称
  game_type ENUM('attention','memory','logic') NOT NULL,
  theme_type ENUM('girl','boy','neutral') NOT NULL,
  description TEXT,                            -- 游戏描述
  min_age INT NOT NULL,                        -- 最小年龄（月）
  max_age INT NOT NULL,                        -- 最大年龄（月）
  difficulty_levels JSON,                      -- 难度级别配置
  resources JSON,                              -- 资源清单
  status ENUM('active','inactive') DEFAULT 'active',
  sort_order INT DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 2.2 游戏关卡表 (game_levels)
```sql
CREATE TABLE game_levels (
  id INT PRIMARY KEY AUTO_INCREMENT,
  game_id INT NOT NULL,
  level_number INT NOT NULL,                   -- 关卡编号（1, 2, 3...）
  level_name VARCHAR(100),                     -- 关卡名称
  difficulty ENUM('easy','medium','hard') NOT NULL,
  config JSON NOT NULL,                        -- 关卡配置（物品数量、时间等）
  unlock_condition JSON,                       -- 解锁条件
  star_requirements JSON,                      -- 星级要求（时间、准确率）
  rewards JSON,                                -- 奖励（金币、勋章）
  status ENUM('active','locked') DEFAULT 'active',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (game_id) REFERENCES game_configs(id),
  UNIQUE KEY (game_id, level_number)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 2.3 游戏记录表 (game_records)
```sql
CREATE TABLE game_records (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,                        -- 玩家用户ID
  child_id INT,                                -- 儿童ID（如果是家长为孩子玩）
  game_id INT NOT NULL,
  level_id INT NOT NULL,
  score INT NOT NULL,                          -- 得分
  stars INT NOT NULL,                          -- 星级（1-3）
  time_spent INT NOT NULL,                     -- 用时（秒）
  accuracy FLOAT,                              -- 准确率
  mistakes INT DEFAULT 0,                      -- 错误次数
  combo_max INT DEFAULT 0,                     -- 最大连击数
  game_data JSON,                              -- 详细游戏数据
  completed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (game_id) REFERENCES game_configs(id),
  FOREIGN KEY (level_id) REFERENCES game_levels(id),
  INDEX idx_user_game (user_id, game_id),
  INDEX idx_completed (completed_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 2.4 成就表 (game_achievements)
```sql
CREATE TABLE game_achievements (
  id INT PRIMARY KEY AUTO_INCREMENT,
  achievement_key VARCHAR(50) UNIQUE NOT NULL,
  achievement_name VARCHAR(100) NOT NULL,
  description TEXT,
  icon_url VARCHAR(500),                       -- 成就图标
  category ENUM('beginner','advanced','master','special') NOT NULL,
  condition JSON NOT NULL,                     -- 解锁条件
  reward JSON,                                 -- 奖励
  sort_order INT DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 2.5 用户成就表 (user_achievements)
```sql
CREATE TABLE user_achievements (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  child_id INT,
  achievement_id INT NOT NULL,
  unlocked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  progress INT DEFAULT 0,                      -- 当前进度
  is_notified TINYINT DEFAULT 0,              -- 是否已通知
  FOREIGN KEY (user_id) REFERENCES users(id),
  FOREIGN KEY (achievement_id) REFERENCES game_achievements(id),
  UNIQUE KEY (user_id, child_id, achievement_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 2.6 游戏设置表 (game_user_settings)
```sql
CREATE TABLE game_user_settings (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  child_id INT,
  bgm_volume FLOAT DEFAULT 0.5,               -- 背景音乐音量（0-1）
  sfx_volume FLOAT DEFAULT 0.8,               -- 音效音量（0-1）
  voice_volume FLOAT DEFAULT 1.0,             -- 语音音量（0-1）
  difficulty_preference ENUM('auto','easy','medium','hard') DEFAULT 'auto',
  animation_speed FLOAT DEFAULT 1.0,          -- 动画速度
  show_hints TINYINT DEFAULT 1,               -- 是否显示提示
  vibration_enabled TINYINT DEFAULT 1,        -- 震动反馈
  settings JSON,                              -- 其他设置
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES users(id),
  UNIQUE KEY (user_id, child_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 🔊 五、音频系统设计

### 5.1 音频分层管理

#### 第一层：背景音乐（BGM）
- **循环播放**: 无缝loop
- **淡入淡出**: 场景切换时crossfade（2秒）
- **音量控制**: 0-1，可调节
- **暂停时**: 淡出（1秒）

**不同游戏的BGM风格**:
- 🌸 公主花园：轻快钢琴，小提琴，梦幻音色
- 🚀 太空冒险：电子合成器，科技感音效
- 🍎 水果记忆：活泼轻快，木琴，清新

#### 第二层：语音提示（Voice）
- **优先级**: 最高（自动降低BGM和SFX音量）
- **播放时机**: 
  - 游戏开始：说明规则
  - 找到目标：鼓励语音
  - 犯错：温柔提示
  - 完成关卡：庆祝语音
- **AI生成**: 温柔女声，富有感情

### 5.2 音频控制器设计

**功能**:
- ✅ 三路独立控制（BGM、语音、音效）
- ✅ 预加载机制（避免卡顿）
- ✅ 播放队列（语音排队播放）
- ✅ 自动混音（语音播放时降低BGM音量）
- ✅ 本地缓存（减少加载时间）

**实现**（audioManager.ts）:
```typescript
class AudioManager {
  // 音频池
  private bgmPool: Map<string, HTMLAudioElement>
  private voiceQueue: Array<{ audio: HTMLAudioElement, priority: number }>
  private sfxPool: Map<string, HTMLAudioElement[]>
  
  // 音量控制
  private bgmVolume = 0.5
  private voiceVolume = 1.0
  private sfxVolume = 0.8
  
  // 播放BGM（循环）
  playBGM(key: string, fadeIn = true): void
  
  // 播放语音（自动排队）
  playVoice(key: string, priority = 0): Promise<void>
  
  // 播放音效（支持多个同时播放）
  playSFX(key: string, volume?: number): void
  
  // 预加载
  preload(urls: string[]): Promise<void>
  
  // 全局静音
```


- **对象池**: 复用游戏对象（避免频繁创建销毁）

- **纹理压缩**: 使用WebP格式
- **LOD**: 远处物体使用低精度模型
- **剔除**: 屏幕外物体不渲染

- **事件解绑**: 避免内存泄漏
