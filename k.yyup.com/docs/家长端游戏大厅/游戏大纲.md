# 儿童脑力训练游戏中心 - 设计大纲

## 📋 总体规划

**位置**: 家长端侧边栏 "游戏中心"  
**目的**: 日常脑力训练，寓教于乐  
**年龄**: 3-6岁儿童  
**特色**: 
- AI生成精美配图
- 语音引导和鼓励
- 性别偏好设计
- 难度自适应
- 成长数据追踪

## 🎯 一、专注力训练游戏（3个）

### 1. 🌸 找不同公主花园（女孩偏好）
**游戏类型**: 2D图片对比  
**适合年龄**: 3-6岁  
**性别倾向**: 👧 女孩喜爱 ★★★★★

#### 游戏玩法
- 展示两幅美丽的公主花园场景
- 场景包含：公主、城堡、花朵、蝴蝶、小动物
- 玩家需要找出5-10处不同之处
- 点击不同处会有闪光特效和鼓励语音

#### 难度分级
- **简单** (3-4岁): 5处不同，大物体差异（颜色、大小）
- **中等** (4-5岁): 7处不同，中等差异（位置、数量）
- **困难** (5-6岁): 10处不同，细微差异（表情、细节）

#### AI资源需求
- **配图**: 生成多组公主花园场景（粉色系、梦幻风格）
  - 主题：公主宫殿、魔法花园、梦幻城堡
  - 提示词：`可爱公主在魔法花园里，粉色城堡，蝴蝶飞舞，扁平插画风格，温馨梦幻`
- **语音**: 
  - 开始："小公主，请仔细观察两幅图片，找出它们的不同哦！"
  - 找对："太棒了！你的眼睛真厉害！"
  - 找错："再仔细看看，你一定能找到的！"
  - 完成："恭喜你！公主花园的秘密都被你发现了！"

#### 技术实现
- Canvas 2D渲染
- 点击检测（热区判定）
- 粒子特效（星星、闪光）
- 计时和评分系统

### 2. 🚀 太空寻宝大冒险（男孩偏好）
**游戏类型**: 2.5D场景探索  
**适合年龄**: 3-6岁  
**性别倾向**: 👦 男孩喜爱 ★★★★★

#### 游戏玩法
- 在太空场景中寻找隐藏的宝藏和外星生物
- 场景包含：宇宙飞船、星球、外星人、能量宝石
- 玩家需要在限定时间内找到所有目标物品
- 找到物品会有激光特效和能量音效

#### 难度分级
- **简单** (3-4岁): 5个大型物品，颜色鲜明，90秒
- **中等** (4-5岁): 8个中型物品，有遮挡，60秒
- **困难** (5-6岁): 12个小型物品，复杂背景，45秒

#### AI资源需求
- **配图**: 生成太空场景（蓝紫色系、科技感）
  - 主题：太空站、星际冒险、外星基地
  - 提示词：`宇宙飞船在太空中，炫酷的星球，可爱的外星人，蓝色科技风格，儿童友好`
- **语音**:
  - 开始："宇航员，准备好了吗？让我们在太空中寻找神秘宝藏！"
  - 找到："哇！发现能量宝石！继续加油！"
  - 时间提醒："还剩30秒，快快快！"
  - 完成："任务完成！你是最棒的太空探险家！"

#### 技术实现
- Canvas 2D + CSS 3D transform
- 视差滚动效果
- 倒计时系统
- 粒子特效（星星、能量波）

### 3. 🦁 动物观察员（中性）
**游戏类型**: 2D动态场景观察  
**适合年龄**: 3-6岁  
**性别倾向**: 👧👦 男女通吃 ★★★★★

#### 游戏玩法
- 屏幕上有多种动物在移动（走、跑、跳）
- 语音提示要找的动物特征（如"找出所有在跳的兔子"）
- 玩家需要快速点击正确的动物
- 错误点击会有温和提示，正确点击有欢快音效

#### 难度分级
- **简单** (3-4岁): 3种动物，单一动作，慢速移动
- **中等** (4-5岁): 5种动物，2种动作，中速移动
- **困难** (5-6岁): 8种动物，3种动作，快速移动

#### AI资源需求
- **配图**: 生成各种动物形象（卡通风格）
  - 动物：兔子、小熊、小鸟、小狗、小猫、大象、长颈鹿、熊猫
  - 提示词：`可爱的{动物}在跳跃/奔跑/走路，Q版卡通风格，明亮色彩，白色背景`
- **语音**:
  - 任务："请找出所有在{动作}的{动物}！"
  - 正确："太棒了！找对了！"
  - 错误："哎呀，再看看！"
  - 完成："太厉害了！你观察得真仔细！"

#### 技术实现
- Canvas 2D动画
- 精灵移动系统
- 碰撞检测
- 动画状态机

- **中等** (4-5岁): 6个水果，序列长度5-8
- **困难** (5-6岁): 9个水果，序列长度8-12

**游戏类型**: 2D拖拽分类  
**适合年龄**: 3-6岁  
**性别倾向**: 👧 女孩喜爱 ★★★★★

- 房间里散落着各种物品
- 玩家需要把物品拖拽到正确的房间/位置
- 房间类别：卧室、厨房、客厅、浴室

- **中等** (4-5岁): 3个房间，12个物品，需思考
- **困难** (5-6岁): 4个房间，16个物品，复杂分类

#### AI资源需求
- **配图**: 生成房间场景和物品（温馨家居风格）

  - 提示词：`可爱的{物品}，扁平插画风格，粉色系，居中构图，白色背景`
- **语音**:
  - 开始："帮娃娃把家里整理干净吧！"
  - 正确："太好了！{物品}应该放在{房间}！"
  - 错误："{物品}好像不应该放在这里哦~"
  - 完成："哇！房子变得整整齐齐，太棒了！"

#### 技术实现
- 拖拽系统（Drag & Drop API）

## 🎮 四、通用系统设计

### 1. 成就系统
- **星级评分**: 根据表现给予1-3星
- **勋章收集**: 完成特定目标获得勋章
  - 🏆 初级玩家（玩10次）
  - 🏆 进阶玩家（玩50次）
  - 🏆 大师玩家（玩100次）
  - 🏆 完美主义者（10次满分）
  - 🏆 快速通关（30秒内完成）

### 2. 成长追踪
- **游戏记录**: 每次游戏的得分、用时、难度
- **能力雷达图**: 专注力、记忆力、逻辑思维三维展示
- **进步曲线**: 按周/月展示进步趋势
- **家长报告**: 每周推送孩子的游戏分析

### 3. 家长陪玩模式
- **双人模式**: 家长和孩子一起玩
- **竞技模式**: 家长VS孩子
- **合作模式**: 一起完成挑战
- **语音鼓励**: 家长可以录制自己的鼓励语音

### 4. 难度自适应
- **AI难度调整**: 根据孩子表现自动调整难度
- **个性化推荐**: 推荐最适合当前能力的游戏
- **学习曲线优化**: 确保既有挑战又不挫败

## 📊 五、技术架构

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **游戏引擎**: Canvas 2D / Three.js (3D游戏)
- **动画**: GSAP / Anime.js
- **音效**: Howler.js
- **状态管理**: Pinia

### 后端API
- **游戏记录**: POST /api/games/records
- **成就解锁**: POST /api/games/achievements
- **排行榜**: GET /api/games/leaderboard
- **AI难度**: POST /api/games/difficulty-adjust

### AI资源生成
- **图片生成**: AIBridge T2I（Doubao Seedream 3.0）
  - 批量生成所有游戏素材
  - 按主题分类存储
  - 自动裁剪和优化
- **语音生成**: AIBridge TTS（Volcengine TTS）
  - 生成所有游戏语音提示
  - 温柔女声，亲切鼓励
  - 按游戏和场景分类
