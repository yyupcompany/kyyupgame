# 招生中心页面设计方案

## 📋 设计概览

**目标**: 使用通用组件库创建统一的招生中心工作台，整合所有招生相关功能到一个中心化页面。

**设计原则**:
- 🎯 **中心化**: 所有招生功能集中在一个页面
- 🔄 **标签页导航**: 使用TabContainer组件管理不同功能模块
- 📊 **数据驱动**: 统计数据和图表展示招生全貌
- 🚀 **高效操作**: 快速访问常用功能和操作

## 🏗️ 页面结构设计

### 整体布局
```
┌─────────────────────────────────────────────────────────────┐
│ CenterContainer: 招生中心                                    │
├─────────────────────────────────────────────────────────────┤
│ 面包屑: 首页 > 招生中心                                      │
│ 操作按钮: [新建计划] [导入数据] [导出报表]                    │
├─────────────────────────────────────────────────────────────┤
│ TabContainer: 标签页导航                                     │
│ [概览] [计划管理] [申请管理] [咨询管理] [数据分析] [AI助手]    │
├─────────────────────────────────────────────────────────────┤
│ 标签页内容区域                                               │
│ (根据选中标签显示不同内容)                                   │
└─────────────────────────────────────────────────────────────┘
```

## 📑 标签页功能设计

### 1. 概览 (Overview)
**目标**: 提供招生全局视图和关键指标

**组件使用**:
- `StatCard` × 4: 核心统计指标
- `ChartContainer` × 2: 趋势图表
- `ActionToolbar`: 快速操作

**内容布局**:
```
┌─────────────────────────────────────────────────────────────┐
│ 统计卡片区域 (4个StatCard)                                   │
│ [总咨询数] [已报名] [试听中] [转化率]                         │
├─────────────────────────────────────────────────────────────┤
│ 图表区域 (2个ChartContainer)                                 │
│ [招生趋势图]              [来源渠道分析]                      │
├─────────────────────────────────────────────────────────────┤
│ 快速操作区域 (ActionToolbar)                                 │
│ [新建计划] [查看申请] [AI分析] [导出报表]                     │
└─────────────────────────────────────────────────────────────┘
```

**数据指标**:
- 总咨询数: 当前活跃咨询数量
- 已报名: 已提交申请的数量
- 试听中: 正在试听阶段的数量  
- 转化率: 从咨询到报名的转化百分比

### 2. 计划管理 (Plans)
**目标**: 管理招生计划的创建、编辑、配额分配

**组件使用**:
- `DataTable`: 计划列表展示
- `FormModal`: 新建/编辑计划
- `DetailPanel`: 计划详情和配额管理
- `ActionToolbar`: 操作工具栏

**内容布局**:
```
┌─────────────────────────────────────────────────────────────┐
│ ActionToolbar: [新建计划] [批量操作] [搜索] [筛选] [导出]     │
├─────────────────────────────────────────────────────────────┤
│ DataTable: 招生计划列表                                      │
│ 列: 计划名称 | 年度 | 学期 | 目标人数 | 已报名 | 状态 | 操作  │
├─────────────────────────────────────────────────────────────┤
│ DetailPanel: 选中计划的详细信息                              │
│ - 基本信息: 计划详情、时间安排                               │
│ - 配额管理: 各班级配额分配                                   │
│ - 进度跟踪: 招生进度和统计                                   │
└─────────────────────────────────────────────────────────────┘
```

**数据来源**: 
- `enrollment_plans` 表
- `enrollment_quotas` 表
- `enrollment_plan_classes` 表

### 3. 申请管理 (Applications)
**目标**: 管理入园申请的审核、面试、录取流程

**组件使用**:
- `DataTable`: 申请列表
- `DetailPanel`: 申请详情和材料审核
- `FormModal`: 面试安排、状态更新
- `ActionToolbar`: 批量操作

**内容布局**:
```
┌─────────────────────────────────────────────────────────────┐
│ ActionToolbar: [批量审核] [安排面试] [搜索] [状态筛选]        │
├─────────────────────────────────────────────────────────────┤
│ DataTable: 申请列表                                          │
│ 列: 申请编号 | 学生姓名 | 家长 | 申请计划 | 状态 | 申请时间   │
├─────────────────────────────────────────────────────────────┤
│ DetailPanel: 申请详情                                        │
│ - 学生信息: 基本资料、家庭信息                               │
│ - 申请材料: 文件上传、审核状态                               │
│ - 流程记录: 审核历史、面试记录                               │
└─────────────────────────────────────────────────────────────┘
```

**数据来源**:
- `enrollment_applications` 表
- `enrollment_application_materials` 表
- `enrollment_interviews` 表

### 4. 咨询管理 (Consultations)
**目标**: 管理招生咨询、跟进记录、转化跟踪

**组件使用**:
- `DataTable`: 咨询记录列表
- `DetailPanel`: 咨询详情和跟进记录

```
┌─────────────────────────────────────────────────────────────┐
│ 统计卡片: [今日咨询] [待跟进] [本月转化] [平均响应时间]       │
├─────────────────────────────────────────────────────────────┤
│ ActionToolbar: [新建咨询] [批量分配] [搜索] [来源筛选]        │
├─────────────────────────────────────────────────────────────┤
│ DataTable: 咨询列表                                          │
│ 列: 咨询编号 | 家长姓名 | 联系方式 | 来源 | 状态 | 负责人     │
├─────────────────────────────────────────────────────────────┤
│ DetailPanel: 咨询详情                                        │
│ - 基本信息: 联系方式、意向信息                               │
│ - 跟进记录: 沟通历史、下次跟进                               │
│ - 转化分析: 意向度评分、转化预测                             │
└─────────────────────────────────────────────────────────────┘
```

**数据来源**:
- `enrollment_consultations` 表
- `enrollment_plan_tracking` 表

### 5. 数据分析 (Analytics)
**目标**: 提供深度数据分析和招生洞察

**组件使用**:
- `ChartContainer` × 4: 多维度图表分析
- `StatCard`: 关键指标对比
- `ActionToolbar`: 时间筛选、导出功能

**内容布局**:
```
┌─────────────────────────────────────────────────────────────┐
│ ActionToolbar: [时间筛选] [维度选择] [导出报表] [刷新数据]    │
├─────────────────────────────────────────────────────────────┤
│ 图表区域 (2×2网格)                                           │
│ [招生趋势分析]            [来源渠道效果]                      │
│ [转化漏斗分析]            [地域分布分析]                      │
├─────────────────────────────────────────────────────────────┤
│ 对比指标 (StatCard)                                          │
│ [同比增长] [环比增长] [目标完成率] [平均获客成本]             │
└─────────────────────────────────────────────────────────────┘
```

**数据来源**:
- 统计API接口
- 多表关联查询
- 时间序列数据

**组件使用**:
- `ChartContainer`: AI分析结果展示
- `StatCard`: 预测指标
- `ActionToolbar`: AI功能选择
- `DetailPanel`: 策略建议详情

**内容布局**:
```
┌─────────────────────────────────────────────────────────────┐

│ AI分析结果                                                   │
│ [预测图表]                [策略建议]                         │
```

## 🔗 API接口设计

### 核心API端点
```typescript
// 概览数据
GET /api/enrollment/overview
// 招生计划
GET /api/enrollment-plans
POST /api/enrollment-plans
PUT /api/enrollment-plans/:id
DELETE /api/enrollment-plans/:id
// 申请管理
GET /api/enrollment-applications
PUT /api/enrollment-applications/:id/status
// 咨询管理  
GET /api/enrollment-consultations
POST /api/enrollment-consultations
// 统计分析
GET /api/enrollment-statistics
// AI功能
GET /api/enrollment-ai/forecast
GET /api/enrollment-ai/strategy
```

## 🎨 用户体验设计

### 导航体验
- 默认显示"概览"标签页
- URL同步: `/enrollment?tab=overview`
- 面包屑导航支持快速返回

### 数据刷新
- 自动刷新: 每5分钟更新统计数据
- 手动刷新: 工具栏刷新按钮
- 实时更新: WebSocket推送重要状态变更

### 响应式适配
- 桌面端: 完整功能展示
- 平板端: 优化布局，保持核心功能
- 移动端: 简化界面，突出关键操作
