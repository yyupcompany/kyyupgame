## 🚀 快速启动

### 1. 启动服务
```bash
# 启动前后端服务
npm run start:all

# 或分别启动
npm run start:frontend  # 前端 (5173)
npm run start:backend   # 后端 (3000)
```

### 2. 初始化数据
```bash
npm run seed-data:complete
```

### 3. 打开浏览器
- 访问: `http://localhost:5173`
- 使用admin账户登录

### 4. 进入AI助手
- 点击顶部导航栏"YY-AI"按钮
- 侧边栏打开，显示AI助手界面
