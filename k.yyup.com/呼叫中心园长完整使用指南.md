## 🚀 快速开始

### 1. 访问系统
```
http://localhost:5173/centers/call-center
```

### 2. 发起呼叫
1. 点击"发起通话"按钮
2. 输入电话号码（例如：18611141133）
3. 输入客户姓名（可选）
4. 选择分机号（可选）
5. 点击"呼叫"

### 3. 通话过程
- 系统自动拨打电话
- 客户接听后，AI自动开始对话
- 系统实时识别客户语音
- AI生成专业回复并播放
- 通话自动记录

### 4. 查看记录
- 实时显示对话文字
- 显示AI回复内容
- 显示通话时长和状态

## 📊 话术管理

### 访问话术管理
```
http://localhost:5173/centers/script-templates
```

### 创建话术
1. 点击"新建话术"
2. 选择分类（问候、介绍、问答、邀约、结束、其他）
3. 输入话术内容
4. 设置关键词
5. 设置优先级
6. 保存

### 话术分类建议

**问候语** (优先级 9-10)
```
"您好，我是XX幼儿园的招生顾问，请问您有时间吗？"
```

**介绍话术** (优先级 9-10)
```
"我们是一所专业的幼儿园，拥有先进的教学设施和经验丰富的教师团队。"
```

**常见问答** (优先级 7-8)
```
Q: "你们的学费是多少？"
A: "我们的学费根据班级和课程不同而有所差异，欢迎您来参观了解。"
```

**邀约话术** (优先级 7-8)
```
"您方便这周末来参观一下吗？我们有专业的讲解员为您介绍。"
```

**结束语** (优先级 5-6)
```
"感谢您的时间，期待为您服务。再见！"
```

## 🔧 故障排除

### 问题1：呼叫失败
**症状**: 显示"呼叫失败"

**解决方案**:
1. 检查电话号码格式
2. 确认SIP服务器配置
3. 检查网络连接
4. 查看系统日志

### 问题2：无法识别语音
**症状**: ASR识别失败

**解决方案**:
1. 检查音频质量
2. 确认麦克风工作正常
3. 调整音量
4. 重新尝试

### 问题3：AI回复不自然
**症状**: AI回复生硬或不合适

**解决方案**:
1. 优化系统提示词
2. 更新话术库
3. 调整AI参数
4. 收集用户反馈

### 问题4：通话中断
**症状**: 通话突然断开

**解决方案**:
1. 检查网络连接
2. 检查防火墙设置
3. 重启系统
4. 联系技术支持
