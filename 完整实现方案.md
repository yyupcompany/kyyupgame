# 🎯 活动海报完整实现方案

## 📊 问题分析

### 现状
- ❌ 预览海报缺少活动关键信息
- ❌ 营销功能未在海报中展示
- ❌ 缺少PC/移动端双预览
- ❌ 没有统一的营销组件库

### 目标
- ✅ 显示完整活动信息（时间、地点、费用、人数）
- ✅ 集成营销功能展示（团购、积赞、优惠券、推荐）
- ✅ PC/移动端双预览
- ✅ 使用全局设计令牌
- ✅ 响应式设计

---

## 🏗️ 架构设计

```
ActivityPosterPreview (主组件)
├─ 活动信息展示
│  ├─ 标题
│  ├─ 时间
│  ├─ 地点
│  ├─ 人数
│  └─ 费用
├─ MarketingSection (营销区域)
│  └─ MarketingBadge × 4 (营销徽章)
│     ├─ 团购
│     ├─ 积赞
│     ├─ 优惠券
│     └─ 推荐奖励
└─ 双预览
   ├─ PC端预览
   └─ 移动端预览
```

---

## 📦 新增组件

### 1. MarketingBadge.vue (单个营销徽章)
- 支持4种类型
- 自定义图标和文本
- 渐变背景和悬停效果
- 响应式设计

### 2. MarketingSection.vue (营销区域)
- 聚合所有启用的营销功能
- 网格布局
- 自动响应式列数
- 空状态处理

### 3. ActivityPosterPreview.vue (完整海报预览)
- 活动信息卡片
- 营销功能集成
- PC/移动端双预览
- 多主题支持

## 🔧 集成步骤

### 步骤1：在ActivityCreate.vue中集成

**位置**: `unified-tenant-system/client/src/pages/activity/ActivityCreate.vue`

在第4步（预览发布）添加：

```vue
<ActivityPosterPreview
  :activity-title="activityForm.title"
  :description="activityForm.description"
  :activity-time="`${formatDate(activityForm.startTime)} ${formatTime(activityForm.startTime)} - ${formatTime(activityForm.endTime)}`"
  :location="activityForm.location"
  :capacity="activityForm.capacity"
  :fee="activityForm.fee"
  :school-name="kindergartenInfo.name"
  :logo-url="kindergartenInfo.logoUrl"
  :phone="kindergartenInfo.phone"
  :address="kindergartenInfo.address"
  :marketing-config="marketingConfig"
  :theme="posterTheme"
/>
```

### 步骤2：在ActivityDetail.vue中集成

**位置**: `unified-tenant-system/client/src/pages/activity/ActivityDetail.vue`

替换现有的海报显示：

```vue
<ActivityPosterPreview
  :activity-title="activity.title"
  :description="activity.description"
  :activity-time="`${formatDate(activity.startTime)} - ${formatDate(activity.endTime)}`"
  :location="activity.location"
  :capacity="activity.capacity"
  :fee="activity.fee"
  :school-name="kindergartenInfo.name"
  :logo-url="kindergartenInfo.logoUrl"
  :phone="kindergartenInfo.phone"
  :address="kindergartenInfo.address"
  :marketing-config="activity.marketingConfig"
/>
```

### 步骤3：在ActivityCenter.vue中集成

**位置**: `unified-tenant-system/client/src/pages/centers/ActivityCenter.vue`

在活动列表中添加预览功能：

```vue
<el-dialog
  v-model="posterPreviewVisible"
  title="海报预览"
  width="90%"
>
  <ActivityPosterPreview
    :activity-title="selectedActivity.title"
    :description="selectedActivity.description"
    :activity-time="`${selectedActivity.startTime} - ${selectedActivity.endTime}`"
    :location="selectedActivity.location"
    :capacity="selectedActivity.capacity"
    :fee="selectedActivity.fee"
    :marketing-config="selectedActivity.marketingConfig"
  />
</el-dialog>
```
